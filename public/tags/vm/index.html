Chronos VM

This virtual machine (VM) is custom-designed for a specific purpose. It includes standard instructions for data movement and arithmetic operations, along with unique jump instructions that extend the capabilities of typical jumping behavior. The VM also has a limited set of flags that can be expanded based on specific needs. It operates using 32-bit opcodes and registers. This VM is created as a part of a challenge for an International CTF, bi0s CTF 2024.
