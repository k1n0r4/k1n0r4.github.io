9:T1ef7,<h3>Overview</h3>
<p>This VM has been completely customized with the intention of using it in a CTF competition. It makes use of 32-bit opcodes and registers. It comes with unique versions for jump instructions and a finite quantity of flags.</p>
<h3>Registers</h3>
<h4>General Registers</h4>
<p>700: S0
701: S1
702: S2
703: S3
704: S4
705: S5
706: S6
707: S7</p>
<h4>Special Registers</h4>
<p>708: MP - Memory pointer
PC - Program counter
SP - Stack pointer
BP - Base pointer</p>
<h5>Flags</h5>
<ul>
<li>EF - Exit Flag</li>
<li>ZF - Zero Flag</li>
</ul>

<h3>Application</h3>
<p>This VM is created as a part of a challenge for an International CTF, bi0s CTF 2024
It is being used here to perform bit manipulation</p>
<h4>Control Flow</h4>
<ol>
<li>Prompt for input</li>
<li>Take input and format it</li>
<li>Store the modified input in memory</li>
<li>Store the same in the stack</li>
<li>Convert into binary and store it in stack and remove the input from memory</li>
<li>Classified array is computed and stored in memory</li>
</ol>
